# wangpq-react-mobx-demo
  使用create-react-app搭建，状态管理使用mobx，ajax请求数据使用axios，示例旨在演示mobx的基本用法，帮助初学者搭建环境和学习
  
# 如何运行

一、下载项目到本地
```bash
git clone https://github.com/wangpq/wangpq-react-mobx-demo.git
```

二、cd 进入项目根目录, 下载依赖
```bash
npm install 
```

三、运行模拟的后台数据
```bash
npm run mock 
```

四、运行项目
```bash
npm run start 
```
